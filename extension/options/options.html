Settings

Gmail Sorter runs classification locally, through your own Ollama server. Your email never leaves your machine.

Required first-time setup

Allow this extension to reach Ollama

Chrome blocks extension requests to localhost:11434 unless Ollama is started with a permissive origin. Run Ollama with:

OLLAMA_ORIGINS=chrome-extension://* ollama serve

Local model

Triage rules

Rewrite these in your own words. The classifier reads them on every email and picks one action from: Star, Archive, Mark read, Leave alone.

Scope

Account

Gmail Sorter uses Google's built-in OAuth. To switch accounts, revoke access and classify again.
